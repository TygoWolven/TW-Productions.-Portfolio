## Installatie

### 🧑‍🤝‍🧑 Clone het project
````
https://github.com/TygoWolven/TW-Productions.-Portfolio.git
````

### 📦️ Installeer alle npm-pakketten
````
npm install
````

### 💿️ Start je developer server
````
npm run dev
````
